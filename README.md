# XocoBot
Telegram bot using telegraf

# Deploy
- Run npm install
- Create a user.json like this: ```{ }```
- Create a botkey.json like this: ```{ "botToken": "yourTelegramBotToken" }```

## user.json
store all registered user -> "chadId": "FirstName"

## botkey.json
store the bot token -> "botToken": "yourTelegramBotToken"
